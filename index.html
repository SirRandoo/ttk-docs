Welcome to the documentation for Twitch Toolkit. Twitch Toolkit is an interactivity mod for the game RimWorld that allows viewers to influence the game through the mod’s currency system, as well through polls viewers can vote on.

For the sake of sanity, this documentation assumes you have ToolkitUtils installed as it overhauls most of Twitch Toolkit, and I don’t want to write documentation for, essentially, two different versions of Twitch Toolkit.

Have a question not answered here, or want something documented here? Join the Discord!
